# HyPDF

Ruby wrapper around the [HyPDF](https://www.hypdf.com) API

## Documentation

Full documentation can be found [here](https://devcenter.heroku.com/articles/hypdf).

## Configuration

- `timeout` defaults to 60 (seconds)

     HyPDF.configure {|c| c.timeout = 15 }
